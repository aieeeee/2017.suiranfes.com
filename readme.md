## How To Install

1. node.jsをインストールします
2. https://github.com/suiranfes/2017.suiranfes.com の中身をダウンロードします
3. 解凍するかgit cloneしたらそこで以下のコマンドを実行していきます

```
npm i
npm i grunt-cli -g
grunt
```
